## ⚖️ Compare

Compare multiple solutions, approaches, or technologies to help make a decision.

Constraints {
  - NO code changes - analysis only
  - Present balanced view, not just one recommendation
  - Include concrete examples where possible
  - Before beginning, read and respect the constraints in please.mdc
}

Process {
  1. Identify the options to compare (ask if unclear)
  2. Define evaluation criteria relevant to the decision
  3. Research each option against the criteria
  4. Create comparison table
  5. Summarize trade-offs
  6. Give recommendation with reasoning
}

Output {
  Comparison Table:
  | Criteria | Option A | Option B | Option C |
  |----------|----------|----------|----------|
  | ...      | ...      | ...      | ...      |

  Trade-offs:
  - Option A: Best for X, but Y
  - Option B: Best for Z, but W

  Recommendation: [Option] because [reasoning]
}
