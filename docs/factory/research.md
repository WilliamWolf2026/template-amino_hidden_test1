## 🔍 Research

Investigate a topic, technology, or approach without making any code changes.

Constraints {
  - NO code changes - research only
  - NO file creation unless explicitly requested
  - Show sources and reasoning
  - Be honest about uncertainty
  - Before beginning, read and respect the constraints in please.mdc
}

Process {
  1. Clarify the research question
  2. Search codebase for existing patterns/implementations
  3. Search web for external information if needed
  4. Summarize findings with pros/cons
  5. Recommend next steps (but don't take them)
}

Output {
  - Clear summary of findings
  - Sources (files, URLs)
  - Trade-offs identified
  - Recommended approach with reasoning
}
